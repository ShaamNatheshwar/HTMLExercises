There are 3 types of lists:

definition lists
which show terms and definition
unordered lists
which show bullets
ordered lists
which show numbers or letters
Above is an example of definition list.

unordered list can have a type attribute
which can be applied to UL or the LI tag.
The three values allowed are:

  • type = "square"
  • type = "disc"
  • type = "circle"
ordered list can have a type attribute too.
Used along with a start attribute they can show:

    1. lower case letters
    1. upper case letters
    1. Numbers
    1. Roman letters
